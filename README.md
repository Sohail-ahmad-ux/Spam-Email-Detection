## 🚀 Project Overview

This project is a **Spam Email/SMS Classifier** built using Machine Learning and Natural Language Processing (NLP).  
It classifies messages as:

- 📩 **Ham (Not Spam)**
- 🚨 **Spam**

---

## 🧠 Problem Statement

Spam messages are a major issue in communication systems.  
This project helps automatically detect spam messages using ML techniques.

---

## 📂 Dataset

- Dataset: `spam.csv`
- Columns:
  - `Category` → Label (ham/spam)
  - `Message` → Text message
